Learning is one XOR.

A tunnel run gathers 27 danger bits — places the environment hit a step the seed did not yet know. Collapse compresses them to 16 bits. seed_packet ^= new_seed keeps what worked and flips what failed. This is the exact operation the live mesh runs every loop.

// ported verbatim from spark_spore.c
uint16_t compress_27_to_16(uint32_t tunnel_bits){
    uint16_t seed = tunnel_bits & 0xFFFF;
    seed ^= (tunnel_bits >> 16) & 0x07FF;   // fold upper 11
    if (popcount27(tunnel_bits) >= 8) seed |= 0x8000; // scar-hi parity
    return seed;
}
void self_teach(CellSeed *c){
    c->seed_packet ^= compress_27_to_16(c->tunnel_bits); // the learning op
    flash_write_neg1(c);   // −1 becomes the library
    push_flash(c);         // −1 → 0 → 1 → 3 → 5 → 8
}
